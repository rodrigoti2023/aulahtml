A World Wide Web (WWW), ou simplesmente Web, é um sistema de documentos interligados e hipertextos que é acessado através da internet. A Web foi inventada por Tim Berners-Lee em 1989, enquanto ele trabalhava no CERN, um laboratório de pesquisa nuclear na Suíça.

A ideia original de Berners-Lee era criar um sistema que permitisse que cientistas compartilhassem informações de forma fácil e eficiente. Ele desenvolveu um protocolo chamado HTTP, que permite que os computadores se comuniquem uns com os outros na Web. Ele também desenvolveu um sistema de marcação chamado HTML, que permite que os documentos sejam formatados e exibidos na Web.

O primeiro site da Web foi lançado em 1991. O site era um catálogo de recursos da Web e estava disponível apenas para os funcionários do CERN. Em 1993, Berners-Lee lançou o Mosaic, o primeiro navegador web gráfico. O Mosaic tornou a Web mais acessível ao público em geral e contribuiu para sua popularização.

A Web cresceu rapidamente nos anos seguintes. Em 1995, já havia mais de 10 milhões de sites na Web. Hoje, a Web é uma rede global que conecta bilhões de computadores. A Web é usada para uma variedade de propósitos, incluindo comunicação, entretenimento, educação e negócios.

Alguns marcos importantes na história da World Wide Web:

1989: Tim Berners-Lee inventa a World Wide Web.

1991: O primeiro site da Web é lançado.

1993: O Mosaic, o primeiro navegador web gráfico, é lançado.

1995: Já há mais de 10 milhões de sites na Web.

Hoje: A Web é uma rede global que conecta bilhões de computadores.

A World Wide Web é uma das invenções mais importantes do século XX. A Web mudou a forma como nos comunicamos, aprendemos e fazemos negócios. A Web é uma ferramenta poderosa que tem o potencial de melhorar a vida de todos.
